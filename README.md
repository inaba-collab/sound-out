# 音声出力アプリ

[アプリを使ってみる](https://sound-out-yans3x6w4ccifj2ao9hama.streamlit.app/)

## アプリの概要
このアプリは書き込んだ内容もしくは、読み込んだテキストを音声出力するためのアプリです。
昨今、音声生成はさまざまな場所で使用されています。
pythonではライブラリを使用することで簡単にインストールができ、streamlitを使うことでアプリ上で生成できるように制作しました。

- 音声切り替え
- 音声出力
